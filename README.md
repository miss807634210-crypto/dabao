# 达宝歌单展示网站

这是一个为虚拟唱见主播达宝打造的歌单展示网站，用于展示达宝的通知以及按分类整理的海量歌曲，支持歌曲分类排序、关键字搜索、歌名复制及直播间跳转功能。

## 功能特点

- 🎵 按分类展示歌曲：说唱类、整活类、国风古风类、民谣治愈类、摇滚热血类、流行抒情类、甜蜜情歌类、伤感虐心类、小众特色类
- 🔍 支持歌名关键字搜索（不区分大小写）
- 📋 点击歌曲名自动复制歌名，添加"点歌 "前缀
- 🎲 随机点歌功能，带有惊喜动画和音效
- 📱 响应式设计，适配电脑、平板、手机等不同尺寸设备
- 🌟 美观的界面设计，带有星空背景和玻璃态效果
- 🎤 直播间快速跳转按钮

## 技术栈

- HTML5
- CSS3 (Tailwind CSS v3)
- JavaScript (原生)
- Font Awesome 图标库

## 如何使用

1. 打开 `index.html` 文件即可访问网站
2. 在顶部可以查看达宝的个人信息和通知
3. 使用分类下拉框筛选不同类型的歌曲
4. 在搜索框中输入关键字搜索歌曲
5. 点击任意歌曲名即可复制歌名（自动添加"点歌 "前缀）
6. 点击"随机点歌"按钮，系统会随机选择一首歌曲并复制
7. 点击"前往直播间"按钮可以直接跳转到达宝的直播间

## 本地开发

无需安装任何依赖，直接在浏览器中打开 `index.html` 文件即可运行。

## 自定义歌单

如需修改歌单内容，请编辑 `index.html` 文件中的 `songData` 对象，按照现有格式添加或修改歌曲分类和歌曲名称。

```javascript
const songData = {
    "分类名称": ["歌曲1", "歌曲2", "歌曲3"...],
    // 其他分类...
};
```
